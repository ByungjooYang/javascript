비교와 연산

비교연산자

== (타입을 비교하지는 않음)

alert(1==2) => false
alert(1==1) => true
alert("one" == "one") => true

===(타입까지 비교한다)

alert(1=='1') => true
alert(1==='1') => false
alert(true == 1) => true (숫자 1외 다른 수는 false)
alert(true == '1') => true
alert(true === 1) => false
alert(true === '1') => false

alert(0 === -0) => true
alert(NaN === NaN) => false (NaN은 계산되지 않는 것이라 비교x)

alert(null == undefined) => true
alert(null === undefined) => false
**null은 개발자가 값이 없는 상태를 지정. undefined는 지정되지 않은것

!(반대연산자)

alert(1 != 2) => true
alert(1 != 1) => false

>, <, <=, >=
