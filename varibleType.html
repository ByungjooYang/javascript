유효범위(Scope)

let vscope = 'global'; => 전역변수

function fscope(){
let vscope = 'local'; => 지역변수
alert(vscope);
}
fscope(); => local
전역변수는 어디에서든 오고 지역변수는 함수 중괄호 안에서만 접근 가능
'local'은 fscope() 함수 밖에서는 못쓴다.
var는 이 문법이 파괴되므로 사용하지 말것

정적 유효범위

js는 함수가 선언된 시점에서의 유효범위를 갖는다. 이런 유효범위 방식을 정적 유효범위
혹은 어휘적 유효범위(렉시컬 스코프(Lexical Scope))라고 한다.
사용될 때가 아니고 정의 될 때의 전역변수가 사용된다는 것이다.

let i = 5;
function a(){
let i = 10;
b();
}

function b(){
document.write(i);
}
function b가 사용될 때의 전역변수 10이 아니라 정의될 때 전역변수 5가 나온다.
