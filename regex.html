정규표현식

let pattern = /a/; => a를 찾는다는 뜻
let pattern = new RegExp('a'); => 위와 같다.

메서드들

exec("문자열") => 문자열 추출

pattern.exec('abcde'); => ["a"]
pattern.exec('bcdef'); = > null

test("문자열") => 문자열 확인

pattern.text('abced'); = > true
pattern.text('bcedf'); = > false

match() => exec()와 비슷

let str = "abcdef" str.match(pattern); => ["a"] 없으면 null 이 나온다.

replace() => 치환

str.replace(pattern, 'A'); = > "Abcde" 문자열이 리턴된다. pattern = /a./ => .은 어떤 문자던지 다 온다는 것.
pattern.exec('abcde'); = >["ab"]


옵션 i와 g

i 옵션 => 대소문자를 구분하지 않는다.

let xi = /a/;
"Abcde".match(xi); => null

let oi = /a/i;
"Abced".match(oi); => ["A"]

g 옵션 => global의 약자.

let xg = /a/;
"abcdea".match(xi); => ["a"]

let og = /a/g;
"abced".match(oi); => ["a", "a"]

혼합

let ig = /a/ig;
"AabcdAa".match(ig); = > ["A", "a", "A", "a"]

캡쳐 => 그룹을 지정해 그룹을 이용하는 것

/(\w+)\s(\w+)/
= > 괄호는 그룹을 의미한다. \w는 문자를 의미하며(A~z, 0~9) + 는 앞의 문자가 하나 이상일 경우 유효해지며
 \s는 스페이스(공백)를 의미한다.

let pattern = /(\w+)\s(\w+)/;
let str = "coding everybody";
let result = str.replace(pattern, "$2", "$1");

console.log(result); => everybody, coding

$1은 첫번째 그룹, $2는 두번째 그룹을 말한다.
