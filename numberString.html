숫자와 Math클래스

Math.pow(3,2); //제곱
Math.round(10.6); //반올림
Math.ceil(10.2); //올림
Math.floor(10.2);//내림
Math.abs(-12.2);//절대값
Math.sqrt(9); // 제곱근
Math.random(); //랜덤수(1미만 소수점)
Math.round(100 * Math.random()); //100 이하 숫자

typeof 객체 => 객체의 타입을 알려준다

alert(typeof "1"); => String
alert(typeof 1); => number
"문자열".length = > 3 (문자열 갯수)
"문자열".indexOf("자") = > 1 (해당 문자의 위치를 알려준다. 0부터 시작.)
